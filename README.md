# Treatment Cost Calculator

A web application for UK beauty salons to calculate the true cost and profitability of their treatments.

## Features

- 💰 Calculate true treatment costs including products, staff time, and overheads
- 📊 Visual dashboard showing profit margins and hourly rates
- 🚨 Identify underpriced treatments that are losing money
- 🧮 Price calculator to find optimal pricing based on target hourly rate
- ⚙️ Customizable salon settings (rent, utilities, insurance)
- 💾 Data persists in browser localStorage

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## How It Works

1. Enter your salon's overhead costs (rent, utilities, insurance)
2. Set your opening hours and target hourly rate
3. Add each treatment with:
   - Current price charged
   - Time taken
   - Products used and costs
   - Staff hourly rate
4. See instantly which treatments are profitable and which need price adjustments

## Technology Stack

- Next.js 16
- React 19
- TypeScript
